# Jurnal-Summarizer-app
Final Project untuk Partnership Hactiv8. Aplikasi mendapatkan ringkasan terstruktur dari isi jurnal yang diupload. Aplikasi ini juga menyediakan fitur chat interaktif untuk bertanya seputar isi jurnal.

#Fitur Utama:
- Upload PDF dan ekstraksi teks otomatis
- Ringkasan terstruktur (penulis, judul, metode, latar belakang, hasil/kesimpulan)
- Chat interaktif terkait dokumen
- Simpan ringkasan & chat history sementara
